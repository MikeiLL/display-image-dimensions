Display dimensions of full size images in media library list view, sortable by square pixel size.

== Description ==

= Sort media library images by size =

= How to Use =

Install and activate the plugin, and the Media Library LIST VIEW will include a sortable "Dimensions" column.

== Installation ==

Install Display Full Image Dimensions through "Add Plugin" or some other clever method, and activate it. Find a new column in the Media Library list view.
