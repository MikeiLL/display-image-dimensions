Display dimensions of full size image in media library list view, sortable by size in square pixels.

== Description ==

= Sort media library images by size =

= How to Use =

Install and activate the plugin, and the Media Library will include a sortable "Dimensions" column.

== Installation ==

Install Display Full Image Dimensions through "Add Plugin" or some other clever method, and activate it. Find a new column in the Media Library list view.
